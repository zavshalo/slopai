SlopAI Setup Guide

  1. Start Ollama: ollama serve
  2. Install the extension via chrome://extensions (or edge://extensions / about:debugging for Firefox) → Developer mode → Load unpacked → select this folder.
  3. If the popup shows a warning: click Fix Ollama Access to download a tailored config.yaml. Place it at:
    • Windows: %USERPROFILE%\.ollama\config.yaml
    • macOS/Linux: ~/.ollama/config.yaml
  4. Restart Ollama and press Retry in the popup.

Permissive dev config:

cors:
  allow_origins: ["*"]
listen: "127.0.0.1:11434"
